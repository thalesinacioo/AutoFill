## 📏 Especificações Técnicas Para Todos os Slides

**Requisitos Chrome Web Store:**
- Dimensão: 1280x800px (mínimo) ou maior
- Formato: PNG ou JPG
- Qualidade: Alta (300dpi ideal)
- Safe Area: Deixar ~50px de margem nas bordas

**Tipografia Recomendada:**
- Fonte: Sistema (Arial, Segoe UI, San Francisco)
- Títulos: Bold, 36-48px, Roxo (#5b4fd8)
- Descrições: Regular, 16-20px, Cinza (#6b7280)
- CTAs: Bold, 20-24px, Branco

**Paleta de Cores:**
- Primária: Roxo (#5b4fd8)
- Secundária: Roxo mais claro (#ede9fe)
- Sucesso: Verde (#16a34a)
- Aviso: Laranja (#ea580c)
- Neutro: Cinza (#6b7280, #9ca3af)
- Background: Branco, cinza claro ou degradado roxo

---

## 🎨 Dicas de Design

1. **Mantém Consistência:** Todas as imagens com a mesma paleta de cores
2. **Não Poluir:** Máximo 3 elementos principais por slide
3. **Ícones:** Use ícones limpos e modernos (flaticon.com, heroicons.com)
4. **Texto:** Máximo 2-3 linhas por elemento
5. **Espaçamento:** Bastante espaço branco para respiro visual
6. **Mobile First:** Considere que alguns usuários veem em celular também
7. **Sem Screenshots:** Evite prints de código, use mockups visuais
