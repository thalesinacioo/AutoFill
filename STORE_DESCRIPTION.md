## Descrição Longa para Store

### O que é?
AutoFill Campos é uma extensão Chrome que permite salvar informações pessoais (emails, telefones, CPF, endereço, etc.) e preenchê-las automaticamente em formulários.

### Como Funciona?
✅ **Salve informações** - Adicione seus dados pessoais no popup
✅ **Preencha com um clique** - Clique direito e selecione a entrada
✅ **Atalhos de teclado** - Use Ctrl+Shift+F (ou ⌘+Shift+F no Mac)
✅ **Sugestões inteligentes** - A extensão sugere os dados corretos automaticamente
✅ **Backup de dados** - Exporte e importe seus dados em JSON

### Principais Features
🔒 **Segurança em Primeiro Lugar**
- Bloqueia sites sensíveis automaticamente
- Nunca armazena senhas
- Dados sincronizados apenas via Chrome Sync

⚡ **Ultra Rápido**
- 3 formas de preencher: clique direito, ícone flutuante, atalho
- Detecção automática de campos compatíveis
- Preenchimento em massa (múltiplos campos de uma vez)

🎯 **Inteligente**
- Reconhece campos pelo placeholder, name, id e aria-label
- Aprende o que você digita e sugere salvar
- Busca entre suas entradas em tempo real

📊 **Organizado**
- Gerenciamento fácil de entradas
- Reordenação por drag-and-drop
- Edição rápida de valores

📤 **Flexível**
- Exporte dados para backup
- Importe dados de qualquer arquivo JSON
- Compartilhe dados entre computadores

### Atalhos Úteis
- **Ctrl+Shift+S** - Abrir popup principal
- **Ctrl+Shift+F** - Abrir picker rápido
- **1-9** - Preencher entrada numerada

### Para Quem é?
✔️ Pessoas que preenchem muitos formulários
✔️ Quem quer economizar tempo em cadastros
✔️ Profissionais que usam múltiplas contas
✔️ Qualquer um que se importa com privacidade

### Sobre Privacidade
📍 Seus dados são **SEUS**
- Nenhuma coleta de dados por nossa parte
- Nenhum rastreamento ou telemetria
- Dados sincronizados apenas via Chrome Sync (padrão do Chrome)
- Compatível com sites bloqueados por padrão

### Dados Sincronizados?
Sim! Seus dados são sincronizados entre seus dispositivos via Chrome Sync se você estiver logado no Chrome. Você pode desabilitar isso nas configurações do Chrome se preferir.

## FAQ

**P: Meus dados são seguros?**
R: Sim. Seus dados ficam em seu navegador Chrome e são sincronizados apenas com sua conta do Chrome (se habilitado). Nós não temos acesso aos seus dados.

**P: Funciona em todos os sites?**
R: Funciona na maioria dos sites. Alguns sites sensíveis (OpenAI, Google, YouTube, redes sociais) são bloqueados por padrão por segurança. Você pode desbloquear se desejar.

**P: Funciona com senhas?**
R: Não, por segurança. A extensão ignora campos de senha.

**P: Posso exportar meus dados?**
R: Sim! Clique em "Exportar" no popup para baixar um arquivo JSON.

**P: E se eu trocar de computador?**
R: Exporte seus dados no computador antigo e importe no novo. Ou ative Chrome Sync para sincronizar automaticamente.

**P: Há limite de entradas?**
R: Não há limite técnico. Chrome Sync suporta até 100MB por aplicativo.
